这里是上海科技大学“从零构建大语言模型”系列讲座的相关课件资料。

### 讲座信息介绍
* Lec1：BPE分词系统设计及算法优化
* Lec2：Transformer模型架构及其现代改进
* Lec3：Transformer架构的代码实现
* Lec4：LLM的训练及自回归解码

其中Lec1、3、4为工程实践讲座，涉及了大量代码实现的讲解，主讲人为匡鹏昊；Lec2为基础知识讲座，主要为Lec3、4做重要的基础铺垫，主讲人为洪沐天。
